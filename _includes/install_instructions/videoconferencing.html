Arbeitsplatz vorbereiten

Wie bei anderen Carpentries-Workshops lernen Sie, indem Sie gemeinsam mit den Instructors „coden“. Dazu müssen Sie sowohl das Fenster für das jeweilige Tool, das Sie kennenlernen werden, als auch das Fenster für Zoom geöffnet haben. Um idealerweise beides gleichzeitig im Blick zu haben, empfehlen wir eine der folgenden Optionen:

  • Zwei Monitore: Wenn Sie zwei Monitore haben, können Sie das Tool, das Sie lernen, auf dem einen, und Zoom auf dem anderen einsehen.
  • Zwei Geräte: Wenn Sie keine zwei Monitore, aber ein weiteres Gerät mit ausreichendem Bildschirm wie z. B. ein Tablet haben, können Sie dieses zweite Gerät für Zoom nutzen.
  • Bildschirmteilung: Wenn Sie nur ein Gerät und einen Bildschirm haben, üben Sie, zwei Fenster (Zoom und eines der Tools, die Sie im Workshop verwenden werden) gleichzeitig zu öffnen: Wie können Sie beides am besten auf Ihrem Bildschirm unterbringen? Funktioniert es für Sie am besten, mit einer Tastenkombination zwischen ihnen hin- und herzuschalten? Probieren Sie es im Voraus aus.
Dieser Blog-Post enthält detaillierte Informationen, wie Sie Ihren Bildschirm einrichten können, um den Workshop zu verfolgen.
