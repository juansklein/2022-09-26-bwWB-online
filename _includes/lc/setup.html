Git

Git ist ein Versionsverwaltungssystem zur Nachverfolgung von Änderungen mit Optionen für einfaches Updaten von geteilten oder öffentlichen Code-Versionen auf https://github.com.

Installations-Anweisungen finden Sie in der Lesson. Bei Problemen können Sie sich wenden an Instructor Dorothea Iglezakis: dorothea.iglezakis@ub.uni-stuttgart.de .

Sie benötigen einen Account auf github.com für die Git-Lesson. Legen Sie bitte einen an, wenn Sie noch keinen haben. Basic-Accounts sind kostenlos. Beachten Sie, welche persönlichen Informationen Sie auf GitHub angeben bzw. privat halten wollen. Hinsichtlich Mail-Adresse kann etwa diese Anleitung nützlich sein. Sie benötigen außerdem einen unterstützten Web-Browser.
